BudgetBnB

Budgetbnb is a clone of Airbnb in which users can rent places to stay for a reasonable price. Users can search for air bnbs by location or the name of the airbnb. Users are able to create, change or cancel reservations.

Running Theory

Running Theory is a competitive fitness site that aims to engage the user through a weekly event. Users are able to navigate the site to find local "tracks" that are running paths created by other users.
